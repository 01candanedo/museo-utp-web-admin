### Version

node js: 18.16.1

npm: 9.7.2

Para obtener una copia local del código, clónelo usando git:

```
https://github.com/jsxmark/museo-utp-web-admin.git
cd museo-utp-web-admin
```

Instalar dependencias:

```
npm i
```

Iniciar un servidor web local ejecutando:

```
npm start
```
